# Naming Conventions

## Branch Naming

```bash
# Features
feature/AUTH-123-add-oauth-provider

# Non-urgent fixes (found in QA)
fix/QA-300-wrong-total-calculation

# Urgent staging fixes (release blocker)
bugfix/QA-400-checkout-crash

# Production emergencies
hotfix/PROD-99-fix-payment-crash
```

**Pattern:** `type/TICKET-ID-short-description`
