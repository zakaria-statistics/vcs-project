# Conventions

## Commit Message Format

```
<type>(<scope>): <description>

feat(payment): add Stripe webhook endpoint
fix(auth): prevent token refresh race
test(payment): add refund flow tests
ci(pipeline): add dependency scanning
```

---

## MR Title Format

```
[TICKET-ID] type: description

[AUTH-123] feat: add OAuth2 integration
[PAY-456] fix: handle webhook timeout
[QA-300] fix: correct tax calculation
```
